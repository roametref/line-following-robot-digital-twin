# line-following-robot-digital-twin
Digital Twin simulation of a line following robot using Python

This project models the behavior of a cyber-physical robotic system including:
- IR sensors
- Digital controller
- Motors and wheels
- Encoders
- Battery and energy consumption
- Robot chassis
- Environment interaction

## Features

- Autonomous line following
- Multiple track shapes:
  - Straight line
  - Circle
  - Sinusoidal path
  - Cosine path
  - Deformed ovals
- Real-time trajectory simulation
- Energy monitoring
- Robot path visualization using Matplotlib

## Technologies

- Python
- NumPy
- Matplotlib

## System Architecture

The project follows a component-based Cyber-Physical System (CPS) architecture using:
- Ports
- Components
- Data flow communication
- Discrete-time simulation
